# 属性
## 通用属性
### name/class/id

| attribules |        values                                   |
| ------     |        ------                                 | 
| name       |    名称: 可以重复；`以下属性均不要以数字开头`   |
| class      |    类:   可以重复；`可以将标签划分到多个类中` |
| id         |    id:   不可重复                            |
| title      |    标题： 用于提示信息                          |
| style      |    样式                                          |
